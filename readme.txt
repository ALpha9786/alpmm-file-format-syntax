| Syntax                      | คำอธิบาย                                                                   |
| --------------------------- | -------------------------------------------------------------------------- |
| `app.title[...]`            | ตั้งชื่อแอป                                                                |
| `app.size[...]`             | ตั้งขนาดหน้าต่าง GUI                                                       |
| `text.[...]`                | แสดงข้อความ                                                                |
| `text.style[...]`           | CSS-like style ของข้อความ (`font-size`, `color`, `font-family`)            |
| `input.[...]`               | ช่องกรอกข้อความ                                                            |
| `button.text[...]`          | ปุ่ม                                                                       |
| `button.style[...]`         | CSS-like style ของปุ่ม (`background`, `color`, `font-size`, `font-family`) |
| `checkbox....`              | Checkbox                                                                   |
| `dropdown....`              | Dropdown list                                                              |
| `slider....`                | Slider (ตัวเลขสูงสุด)                                                      |
| `tabview....`               | Tab view                                                                   |
| `onclick.button;alert[...]` | Event ปุ่มกด → แสดง alert                                                  |
| Property                                           | Widget                | Description  |
| -------------------------------------------------- | --------------------- | ------------ |
| `font-size`                                        | Text / Button / Input | ขนาดตัวอักษร |
| `font-family`                                      | Text / Button / Input | Font family  |
| `color`                                            | Text / Button / Input | ตัวอักษร     |
| `background`                                       | Button / Input        | พื้นหลัง     |
| *(อนาคตสามารถเพิ่ม border, padding, margin, etc.)* |                       |              |
